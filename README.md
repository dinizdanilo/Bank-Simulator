# 💳 Sistema de Conta Bancária em Java

Projeto simples desenvolvido para praticar conceitos de programação orientada a objetos em Java.

O sistema simula operações básicas de uma conta bancária via terminal.

## 🚀 Funcionalidades

- Criar uma conta bancária
- Realizar depósito inicial (opcional)
- Fazer depósitos adicionais
- Realizar saques com taxa fixa
- Exibir dados atualizados da conta

## 🧠 Conceitos aplicados

- Encapsulamento
- Construtores sobrecarregados
- Orientação a objetos (POO)
- Validação de dados
- Organização em pacotes (application / entities)
